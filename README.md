# applicativoScreen
Projeto desenvolvido com o objetivo de aplicar conceitos de navegação entre telas utilizando Jetpack Compose.

Funcionalidades
- Tela de login inicial
- Menu principal com navegação
- Tela de perfil
- Tela de pedidos
- Navegação entre telas usando NavController

Estrutura das telas
- O app é dividido nas seguintes telas:
- LoginScreen → Tela inicial com botão de entrada
- MenuScreen → Menu com opções de navegação (Perfil, Pedidos e Sair)
- PerfilScreen → Tela de perfil com botão de voltar
- PedidosScreen → Tela simples com layout básico
